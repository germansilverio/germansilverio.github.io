# Portfolio de Germán

Este es el código fuente de tu nuevo portfolio personal.

## Estructura
- `index.html`: La página principal.
- `css/styles.css`: Estilos visuales (Modo oscuro, diseño moderno).
- `js/main.js`: Lógica para animaciones y navegación.
- `assets/images/`: Carpeta para tus imágenes.

## Cómo usar
1. Abre el archivo `index.html` en tu navegador (Chrome, Edge, etc.) para ver el sitio.
2. Para personalizar:
   - **Imágenes**: Coloca las capturas de tu app de Finanzas en `assets/images/`.
   - **Textos**: Edita `index.html` con cualquier editor de texto (Notepad++, VS Code) para cambiar tu historia o detalles de contacto.
